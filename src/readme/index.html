Manual for S21_SmartCalc_v1.0

S21_SmartCalc_v1.0 is a super cool calculator to calculate anything.

Part 1. Basic Operations

Arithmetic operations:

  • Parentheses (a + b)
  • Addition a + b
  • Subtraction a - b
  • Multiplication a * b
  • Division a / b
  • Power a ^ b
  • Remainder a mod b
  • Unary plus +a
  • Unary minus -a

Functions:

  • cos(x) cosine of x
  • sin(x) sine of x
  • tan(x) tan of x
  • acos(x) arccosine of x
  • asin(x) arcsine of x
  • atan(x) arc tangent of x
  • sqrt(x) of x root
  • ln(x) of x natural logarithm
  • log(x) of x decimal logarithm

How to use these?

  1. Type functions and hit '='
  2. If you enter something separately x
